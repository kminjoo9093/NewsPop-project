### **개발 목표**

- 오픈 api를 이용해 조건에 맞는 다양한 데이터를 요청하고 처리하는 방법 학습<br>
- 그 과정에서 발생하는 에러 상황을 처리하고 사용자에게 적절한 메시지 제공
